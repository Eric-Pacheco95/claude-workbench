# context/

Workspace context files -- lessons, session notes, and reference material generated during Claude Code sessions.

## Subdirectories

| Directory | Contents |
|-----------|----------|
| `teach/` | Lessons generated by `/teach` -- saved for future reference |

## Usage

Files here are written by skills during sessions. They are read-only reference material -- do not edit manually unless correcting errors.

- `/teach <topic>` saves lessons to `teach/{slug}.md`
- Any skill may write session context here for later reference
